# CUT-TAG-analysis

A set of command line tools CUT&Tag data processing and analysis, modified from the [tutorial](https://yezhengstat.github.io/CUTTag_tutorial/) by Zheng Y et al (2020). Protocol.io

## Pre-requisites and dependencies
*sra-toolkit 
*trim-galore 
*bowtie2 
*samtooos 
*bedtools 
*python3-biopython 
*macs2
